Privacy Policy

Effective Date: September 2025

VeilMark provides tools to detect AI-generated content through a Discord bot, a browser extension, and an API service. We take user privacy seriously and are committed to handling data responsibly across all platforms.

1. Information We Collect

  • Discord Bot: Messages, images, guild IDs, and user IDs provided via slash commands or autoscan features.
  • Browser Extension: Text snippets or image URLs explicitly selected by the user for scanning, and locally stored API keys.
  • API: Text or image URLs submitted to the API for analysis, account IDs, and usage statistics (e.g., scan counts).

2. How We Use This Information

  • To analyze content for AI-generated text or images and return results.
  • To enforce usage tiers, limits, and feature access.
  • To improve detection models and service reliability.
  • To respond to feedback, bug reports, or abuse reports.

3. Data Storage

We store limited configuration settings, usage logs, and scan results needed to provide service functionality. We do not permanently store full message content or images beyond what is necessary for immediate analysis and logging. API keys entered in the browser extension are stored locally on the userâ€™s device and are not transmitted elsewhere.

4. Third-Party Services

Some analysis may be routed through external AI providers (e.g., OpenAI) if fallback features are enabled. No user-identifying information is sent to these services beyond the content being analyzed.

5. User Control

  • Discord Bot: Server administrators can configure or disable features via slash commands, or remove the bot at any time.
  • Browser Extension: Users can remove or reset their API keys via the Options page, or uninstall the extension at any time.
  • API: Users can manage or delete their accounts by contacting support.

6. Contact

For questions or concerns about this policy, please contact us at veilmark.bot@gmail.com.

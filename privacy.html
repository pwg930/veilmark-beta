Privacy Policy

Effective Date: August 4, 2025

VeilMark is a Discord bot designed to help communities detect AI-generated messages and images. We take user privacy seriously and are committed to handling data responsibly.

1. Information We Collect

  • Messages and images submitted to VeilMark via slash commands or autoscan features
  • Guild ID and user ID (for usage tracking and moderation features)
  • Configuration settings related to your server's use of VeilMark

2. How We Use This Information

  • To analyze messages and images for AI-generated content
  • To enforce usage tiers, limits, and feature access
  • To improve our detection models and user experience
  • To respond to feedback or bug reports

3. Data Storage

We store data such as guild configuration, limited usage logs, and model inference results. We do not store the full content of messages beyond what is necessary for immediate analysis and logging.

4. Third-Party Services

Some analysis may be routed through external AI services (e.g., OpenAI) if fallback is enabled. No user-identifying information is sent to these services beyond the message or image being analyzed.

5. User Control

Server administrators can enable or disable features using slash commands. You may also remove the bot at any time.

6. Contact

For questions or concerns, please contact us at veilmarkbeta@gmail.com
